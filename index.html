Sistema integral de ventas

Sistema de compra-venta con clientes registrados y precios personalizados.

Detalles del proyecto

Un sistema que maneja la compra-venta de productos con clientes registrados con su DNI, permite ofrecer precios personalizados a cada comprador con cada producto. Los usuarios pueden introducir sus productos mediante un documento .csv.

Discord Bot

Un bot de Discord en Python que combina moderación, entretenimiento e integración con fuentes externas de datos..

Detalles del proyecto

Bot de Discord en Python con discord.py que ofrece comandos de moderación, interacción y reproducción de audio en canales de voz. Permite banear usuarios, enviar saludos, frases, GIFs, números aleatorios y reproducir audios con FFmpeg. Además, integra la API de Reddit con praw para mostrar publicaciones aleatorias y un archivo CSV de Google Sheets para enviar frases externas en Discord.

Herramientas: Python, discord.py, praw
